# v0.3.0 — Enhanced Privacy and UX (2026-02-15)

## Features
- Tier 3 (100 USDC) and Tier 4 (1000 USDC) deposit support
- Intermediate wallet system for enhanced privacy
- Batch transfer support for merchants
- Dark mode theme
- Keyboard shortcuts for power users

## Improvements
- Redesigned dashboard with better overview of balances
- Improved error messages with actionable suggestions
- Faster page loads with code splitting optimization
- Reduced bundle size by 25% through tree shaking

## Bug Fixes
- Fixed proof generation failure on Safari 17
- Resolved issue where deposit notes could be duplicated
- Fixed network switching not updating displayed balances
- Corrected fee calculation rounding errors

## Security
- Added CSP headers for XSS prevention
- Implemented rate limiting per wallet address
- Added input sanitization on all API endpoints
