# v0.1.0 — Privacy Pool MVP (2026-01-15)

## Features
- ZK proof generation in browser via WebAssembly
- USDC deposit into privacy pool (Tier 1 and Tier 2)
- Private transfer with zero-knowledge proof verification
- Deposit note management with local storage backup
- Basic transaction history view

## Technical
- Circom circuits for deposit and withdrawal proofs
- Merkle tree implementation for deposit tracking
- Relayer service for anonymous transaction submission
- WASM build pipeline for in-browser proof generation

## Security
- Input validation on all user-facing forms
- Rate limiting on API endpoints
- Nonce-based wallet authentication

## Known Issues
- Proof generation slow on mobile devices (> 30 seconds)
- No support for Tier 3 and Tier 4 deposits
- Transaction history limited to 50 most recent
