# Wallet Types

## Overview
Understanding different wallet types helps you choose the right security model for your BaseUSDP usage.

## Browser Extension Wallets
Examples: MetaMask, Coinbase Wallet Extension, Rabby

**Pros:**
- Convenient for frequent desktop use
- Quick transaction signing
- Rich ecosystem of supported dApps

**Cons:**
- Security depends on browser and extension isolation
- Vulnerable to phishing attacks
- Private keys stored in browser storage

## Mobile Wallets
Examples: Coinbase Wallet Mobile, MetaMask Mobile, Rainbow

**Pros:**
- Biometric authentication support
- Always available on your phone
- QR code scanning for payments

**Cons:**
- Smaller screen for reviewing transactions
- Phone loss or theft is a risk
- Some wallets have limited Base support

## Hardware Wallets
Examples: Ledger, Trezor (via MetaMask bridge)

**Pros:**
- Private keys never leave the device
- Highest security for large balances
- Physical confirmation of transactions

**Cons:**
- Less convenient for frequent use
- Additional hardware cost
- Slower transaction signing

## Smart Contract Wallets
Examples: Safe (Gnosis Safe), Coinbase Smart Wallet

**Pros:**
- Multi-signature support for shared treasuries
- Social recovery options
- Gasless transactions via paymasters

**Cons:**
- Higher gas costs for some operations
- More complex setup
- Potential compatibility issues with some protocols

## Recommendation
For most BaseUSDP users, a browser extension wallet (MetaMask or Coinbase Wallet) offers the best balance of security and convenience. For large holdings, consider a hardware wallet connected through MetaMask.
