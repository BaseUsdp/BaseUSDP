# What is Base?

## Overview
Base is an Ethereum Layer 2 (L2) network built by Coinbase using the OP Stack. It provides a secure, low-cost, developer-friendly environment for building decentralized applications.

## Key Features
- **Low fees:** Transactions cost fractions of a cent compared to dollars on Ethereum mainnet
- **Fast confirmations:** 2-second block times for near-instant finality
- **Ethereum security:** Transaction data is posted to Ethereum mainnet for security
- **EVM compatible:** Any Ethereum smart contract works on Base without modification

## Why BaseUSDP Chose Base
1. **Cost efficiency:** Privacy transactions involve complex cryptographic operations that would be expensive on mainnet
2. **Speed:** Fast block times enable a smooth user experience for payments
3. **Coinbase ecosystem:** Integration with Coinbase Wallet and the broader Coinbase ecosystem
4. **Growing adoption:** Base has one of the fastest-growing DeFi ecosystems

## Technical Details
- Chain ID: 8453
- Block time: 2 seconds
- Data availability: Ethereum mainnet (via blobs)
- Consensus: Sequencer-based with planned decentralization
- Bridge: Official Base Bridge for ETH and ERC-20 tokens

## Bridging to Base
To use BaseUSDP, you need USDC and a small amount of ETH on Base. You can:
- Bridge from Ethereum using the official Base Bridge
- Transfer from Coinbase directly to Base
- Use a cross-chain bridge like Across or Stargate
