# Bridge Integration

## Getting USDC on Base
- **Official Base Bridge** (bridge.base.org): ETH, USDC. 1-5 min.
- **Circle CCTP**: Native USDC bridging. ~15 min. No bridge fee.
- **Third-party**: Across, Stargate, Hop Protocol
- **Direct**: Coinbase withdrawal to Base, DEX swap on Base

Future: In-app bridge integration for single-transaction deposits from any chain.
