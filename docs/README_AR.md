# BaseUSDP — مدفوعات خاصة على Base

## الوصف
BaseUSDP هو بروتوكول مدفوعات خصوصية مفتوح المصدر مبني على شبكة Base. يتيح تحويلات USDC خاصة باستخدام إثباتات المعرفة الصفرية (ZK proofs)، مما يضمن بقاء تفاصيل المعاملات سرية.

## الميزات الرئيسية
- **تحويلات خاصة:** أرسل USDC دون الكشف عن الارتباط بين المرسل والمستلم
- **إثباتات المعرفة الصفرية:** يتم إنشاء الإثباتات في المتصفح، بدون بيانات خاصة على الخادم
- **تكلفة منخفضة:** المعاملات على Base تكلف أقل من 0.01 دولار
- **أسماء المستخدمين:** أرسل المدفوعات إلى أسماء مقروءة بدلاً من عناوين المحافظ
- **مدفوعات x402:** بروتوكول دفع للتجار مع دعم QR

## البدء السريع
```bash
git clone https://github.com/BaseUsdp/BaseUSDP.git
cd BaseUSDP
npm install
cp .env.example .env.local
npm run dev
```
