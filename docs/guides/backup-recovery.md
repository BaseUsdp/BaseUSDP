# Backup and Recovery

## What to Back Up
Deposit notes — without them, funds cannot be recovered.

## Methods
1. **Browser Export**: Settings > Security > Export Notes (encrypted)
2. **Manual Copy**: Copy note after each deposit, store in password manager
3. **Paper Backup**: Print and store in a safe

## Recovery
Settings > Security > Import Notes > select backup > enter password.

## Best Practices
- Back up after every deposit
- Test recovery periodically
- Keep backups in 2+ separate locations
- Use strong encryption passwords
- Never share notes with anyone
