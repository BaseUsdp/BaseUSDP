# Design System

## Colors
- Primary: #3B82F6 (Blue)
- Background: #0A0B14 (Dark navy)
- Success/Warning/Error semantic colors

## Typography
- Headings & Body: Inter (variable)
- Code: JetBrains Mono

## Spacing
4px grid. Values: 4, 8, 12, 16, 24, 32, 48, 64px.

## Components
Buttons (primary/secondary/ghost), Cards, Inputs, Modals, Toasts.

## Icons
Lucide React. Sizes: 16/20/24px.

## Breakpoints
Mobile (<640px), Tablet (640-1024px), Desktop (>1024px).

See `src/styles/variables.css` for tokens.
