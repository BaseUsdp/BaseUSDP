# Keyboard Shortcuts

## Overview
BaseUSDP supports keyboard shortcuts for power users who prefer navigating the application without a mouse.

## Global Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Open command palette |
| `Ctrl/Cmd + /` | Toggle keyboard shortcut help |
| `Esc` | Close current modal or dialog |

## Navigation
| Shortcut | Action |
|----------|--------|
| `G then D` | Go to Dashboard |
| `G then P` | Go to Deposit page |
| `G then T` | Go to Transfer page |
| `G then S` | Go to Swap page |
| `G then H` | Go to History page |
| `G then E` | Go to Settings |

## Actions
| Shortcut | Action |
|----------|--------|
| `N then D` | New deposit |
| `N then T` | New transfer |
| `N then S` | New swap |

## Wallet
| Shortcut | Action |
|----------|--------|
| `W then C` | Connect wallet |
| `W then D` | Disconnect wallet |
| `W then S` | Switch network |

## Accessibility
All keyboard shortcuts are designed to be accessible and follow WAI-ARIA best practices. Screen reader users can navigate all functionality using standard tab navigation.

## Customization
Keyboard shortcuts can be customized in Settings > Preferences > Keyboard Shortcuts. Custom shortcuts are stored locally in your browser.
