# Wallet Setup Guide

## Overview
BaseUSDP supports multiple wallet providers for interacting with the Base network. This guide walks you through connecting your wallet and configuring it for privacy-preserving payments.

## Supported Wallets
- **MetaMask** — The most popular browser extension wallet
- **Coinbase Wallet** — Native Base network support
- **WalletConnect** — Connect any mobile wallet via QR code
- **Rainbow** — User-friendly mobile wallet with Base support

## Step 1: Install a Wallet
Download and install one of the supported wallets from their official website. We recommend MetaMask for desktop users and Coinbase Wallet for mobile users.

## Step 2: Add Base Network
If your wallet does not automatically detect Base, add it manually:
- **Network Name:** Base
- **RPC URL:** https://mainnet.base.org
- **Chain ID:** 8453
- **Currency Symbol:** ETH
- **Block Explorer:** https://basescan.org

## Step 3: Fund Your Wallet
You need a small amount of ETH on Base for gas fees. Bridge ETH from Ethereum mainnet using the official Base Bridge at bridge.base.org or purchase directly through your wallet provider.

## Step 4: Connect to BaseUSDP
Navigate to the BaseUSDP app and click "Connect Wallet". Select your wallet provider and approve the connection request. Your wallet address will appear in the top-right corner.

## Troubleshooting
- If the wallet does not connect, ensure you are on the Base network
- Clear your browser cache if you experience persistent connection issues
- Check that your wallet extension is up to date
