# Network Switching Guide

## Supported Networks
BaseUSDP currently supports the following networks:
- **Base Mainnet** — Production network for real transactions
- **Base Sepolia** — Testnet for development and testing

## Switching Networks

### Automatic Switching
When you connect to BaseUSDP, the application will prompt you to switch to the correct network if your wallet is connected to an unsupported chain. Click "Switch Network" in the prompt and approve the network change in your wallet.

### Manual Switching
1. Open your wallet extension or mobile app
2. Click the network selector (usually at the top of the wallet)
3. Select "Base" or "Base Sepolia" depending on your needs
4. The BaseUSDP app will automatically detect the network change

- Currency: ETH (testnet)

## Testnet Faucets
For testing on Base Sepolia, you can obtain testnet ETH and USDC from:
- Base Sepolia Faucet: https://www.coinbase.com/faucets/base-ethereum-sepolia
- BaseUSDP Testnet Faucet: Available in the app when connected to Sepolia

## Network Indicators
The BaseUSDP interface displays the current network in the header. A green indicator means you are on mainnet, and an orange indicator means you are on testnet.
