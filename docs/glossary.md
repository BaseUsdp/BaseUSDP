# Glossary

**Anonymity Set** — Group of indistinguishable users.
**Base** — Ethereum L2 by Coinbase (OP Stack).
**Commitment** — Cryptographic hash binding to a hidden value.
**Deposit Note** — Secret key for spending deposited funds.
**Gas** — Transaction processing fee.
**Merkle Tree** — Data structure for efficient set membership proofs.
**Nullifier** — Value preventing double-spending.
**Privacy Pool** — Contract holding deposited USDC.
**Relayer** — Service submitting transactions for user privacy.
**USDC** — USD-pegged stablecoin by Circle.
**WASM** — WebAssembly for browser-based proof generation.
**x402** — HTTP 402-based payment protocol.
**ZK-SNARK** — Succinct zero-knowledge proof system used by BaseUSDP.
