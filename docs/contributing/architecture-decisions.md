# Architecture Decision Records

## ADR-001: Next.js Framework
SSR for SEO, API routes for backend, TypeScript, Vercel integration.

## ADR-002: Client-Side Proof Generation
ZK proofs generated in browser via WASM. Secrets never leave the device.

## ADR-003: Supabase Backend
Rapid development, built-in auth, real-time, managed PostgreSQL.

## ADR-004: Fixed Deposit Tiers
Fixed amounts (1/10/100/1000 USDC) create larger anonymity sets.

## ADR-005: Relayer Submission
Prevents user address appearing as tx sender. Self-relay as fallback.
