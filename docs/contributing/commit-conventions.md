# Commit Conventions

Format: `type(scope): description`

Types: feat, fix, docs, style, refactor, perf, test, build, ci, chore, security

Examples:
- `feat(transfer): add batch transfer support`
- `fix(deposit): correct gas estimation`
- `docs(api): add websocket docs`

Rules: imperative mood, under 100 chars, reference issues in footer.
