# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 0.5.x | Yes |
| 0.4.x | Security fixes only |
| < 0.4 | No |

## Reporting
Email security@baseusdp.com. Do NOT open public issues.
- Acknowledgment: 48 hours
- Assessment: 1 week
- Critical fix: 7 days
- Standard fix: 90 days
